A cloud-deployed secure log analyzer that parses Apache & Nginx server logs using a custom-built MapReduce engine — giving developers instant visibility into traffic patterns, errors, and suspicious activity.
Why This Project

Server logs are goldmines of insight — but raw .log files are unreadable noise. Most tools are either too complex or require expensive infrastructure. LogRadar was built to make log analysis accessible: upload a file, get a clean analytics breakdown instantly, powered by a MapReduce engine written from scratch in pure Python.
